# EP010 / VT-02: OBS色空間・LUT設定の正解 - 601/709、Full/Partialの違いと適切な選択

## 必要な動画素材

### 画面キャプチャ

- OBS設定画面（設定 > 詳細設定 > 色空間・カラーレンジ）
- OBS出力設定画面（配信/録画タブの映像エンコーダー色空間設定）
- NVIDIAコントロールパネルの色設定画面（出力のダイナミックレンジ: Full / Limited）
- OBSソースフィルタ画面（LUTフィルタの適用）
- YouTubeプレイヤー上での Stats for Nerds 表示画面（Color space情報）

### デモ映像

- 色空間設定が不一致な状態のデモ（映像がくすむ / 白飛びする / コントラストが不自然）
- BT.601とBT.709の違いを示すカラーチャート表示の比較映像
- Full RangeとLimited Rangeの違いを示す比較映像（グラデーション表示で黒つぶれ・白飛びの差）
- 正しい色空間設定に修正した後の映像変化デモ
- LUTフィルタ適用前後の比較映像
- GPU側の色設定を変更した場合の影響デモ

### 図解・テロップ素材

- 映像信号の処理フロー図（GPU → OBS → エンコード → 配信先 の各段階での色空間設定箇所）
- BT.601 / BT.709 / BT.2020 の色域範囲を示すCIE色度図
- Full Range（0-255）vs Limited Range（16-235）の階調対応図
- 環境別の推奨設定早見表（ゲーム配信 / カメラ映像 / VTuber）

## 必要機材

### 所有済み

- PC2: NVIDIA GeForce RTX 3060（NVIDIAコントロールパネルでの色出力設定デモ用）
- モニター: 3枚（異なるパネル種別での色再現比較に活用可能）

### 追加購入

なし。本エピソードはソフトウェア設定完結（OBS設定 + NVIDIAコントロールパネル）。
