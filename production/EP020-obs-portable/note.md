# EP020 / CM-02: OBSポータブルモード運用 - レジストリ汚染回避と環境分離

## 必要な動画素材

### 画面キャプチャ

- OBSのショートカットプロパティ画面（起動オプション --portable の追加）
- ポータブルモードで起動したOBSのデータディレクトリ構造（OBSインストールフォルダ直下にconfigが生成される様子）
- 通常モードのOBSデータディレクトリ（%APPDATA%\obs-studio\）との比較
- レジストリエディタ画面（regedit: OBS関連のレジストリキーの有無確認）
- エクスプローラーでのフォルダコピー操作画面

### デモ映像

- OBSのショートカットに --portable オプションを追加する手順デモ
- ポータブルモードで初回起動し、設定ウィザードを完了する操作デモ
- 通常モードとポータブルモードのデータ保存場所の違いを確認するデモ
- 複数のポータブルOBS環境を共存させるデモ（配信用 / 録画用 / テスト用）
- ポータブルOBSフォルダをUSBドライブにコピーして別PCで起動するデモ
- バックアップとしてフォルダごとZIP圧縮する操作デモ

### 図解・テロップ素材

- 通常モード vs ポータブルモードのデータ保存場所比較図
- ポータブルモードのメリット・デメリット一覧表
- 複数環境の運用構成図（用途別にフォルダを分離するイメージ）
- ポータブルモード移行チェックリスト

## 必要機材

### 所有済み

- PC1 / PC2（複数環境でのポータブルOBS起動デモ用）

### 追加購入

なし。本エピソードはソフトウェア設定完結。USBドライブは既存のもので対応可能。
