# EP003 / AE-03: ノイズゲート最適化 - 環境音スペクトル分析に基づく閾値設定

## 必要な動画素材

### 画面キャプチャ

- スペクトルアナライザ（Voxengo SPAN 等）のプラグインUI画面
- OBSノイズゲートフィルタの設定画面（Open/Close Threshold、Attack/Hold/Release）
- OBS音声フィルタの並び順画面（ノイズゲートの推奨配置位置）

### デモ映像

- 無音時の環境音スペクトル計測画面（エアコン、PC ファン等のノイズフロアを表示）
- 発声時のスペクトル計測画面（音声帯域とノイズ帯域の重なりを確認）
- ノイズゲート閾値が高すぎるケースのデモ（声の出だしが途切れる / 小声が拾われない）
- ノイズゲート閾値が低すぎるケースのデモ（環境音やキーボード打鍵音が漏れる）
- 最適な閾値設定後のデモ（自然な発話開始・終了）
- Attack / Hold / Release パラメータの効果を個別に示すデモ

### 図解・テロップ素材

- 環境音と音声のスペクトル重畳図（閾値の決定ポイントを矢印で示す）
- ノイズゲートの開閉動作を示すタイムライン図（信号レベル vs 閾値）
- 推奨パラメータ設定値の一覧表（環境別: 静かな部屋 / エアコン有 / 機械式キーボード有）

## 必要機材

### 所有済み

- マイク: Audio-Technica AT-2020
- オーディオIF: Roland VT-4
- ポップフィルター、ショックマウント、マイクアーム

### 追加購入

なし。本エピソードはソフトウェア完結（Voxengo SPAN等のスペクトルアナライザ + OBSノイズゲートフィルタ）。
