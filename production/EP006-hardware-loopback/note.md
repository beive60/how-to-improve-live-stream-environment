# EP006 / AE-06: Hardware Loopback完全ガイド - DSP内蔵オーディオIFによるハードウェアルーティング

## 必要な動画素材

### 画面キャプチャ

- RME TotalMix FXのミキサー画面（入力 / ソフトウェアプレイバック / ハードウェア出力の3段構成）
- TotalMix FXのルーティングマトリクス画面（Loopback有効化の手順）
- MOTU CueMix / Universal Audio Console 等の代替製品の画面（他製品との比較用）
- OBS側の音声入力設定画面（ループバックチャンネルの選択）

### デモ映像

- オーディオインターフェース本体の外観・背面端子の実写映像
- TotalMix FXでLoopbackチャンネルを有効化する操作手順の画面録画
- ハードウェアルーティングで複数音源（マイク / ゲーム音 / Discord）をミックスするデモ
- DSPエフェクト（EQ / コンプ）をハードウェア側で適用する操作デモ
- ソフトウェアルーティング（VoiceMeeter等）との CPU負荷比較デモ

### 図解・テロップ素材

- ハードウェアルーティング構成図（物理入力 → DSP → Loopback → OBS）
- ソフトウェアルーティングとの比較図（CPU処理 vs DSP処理の違い）
- DSP内蔵オーディオインターフェースの主要製品比較表（RME / MOTU / UA）
- ハードウェアルーティングのメリット・デメリット一覧

## 必要機材

### 所有済み

- マイク: Audio-Technica AT-2020

### 追加購入（必須）

- DSP内蔵オーディオIF（RME Babyface Pro FS / MOTU M4 / Audient iD14 mk II）
  - 本エピソードの主題そのもの。DSP内蔵IFのハードウェアルーティング（TotalMix FX等のDSPミキサー操作）が企画の中核
  - 外観・背面端子の実写撮影が必要
  - RME Babyface Pro FSであればTotalMix FXを実演でき、MOTU/UAとの比較表も制作可能
  - AE-05で購入済みであれば追加購入不要
