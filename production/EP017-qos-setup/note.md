# EP017 / NE-04: QoS設定実践ガイド - ルーターレベルのパケット優先制御

## 必要な動画素材

### 画面キャプチャ

- ルーター管理画面のQoS設定ページ（一般的な家庭用ルーター例: ASUS / TP-Link / NEC 等）
- QoSルール作成画面（アプリケーション指定 / ポート指定 / IPアドレス指定）
- ルーターのトラフィックモニター画面（帯域使用状況のリアルタイム表示）
- Windows側のDSCP設定画面（グループポリシーエディタ / PowerShellコマンド）

### デモ映像

- QoS未設定の状態で、配信中に他デバイスが大容量ダウンロードを開始した場合のデモ（配信がカクつく）
- QoS設定後の同一条件でのデモ（配信の帯域が優先され、安定を維持）
- ルーター管理画面でQoSルールを作成・適用する操作手順デモ
- 優先度の異なる複数デバイスが同時に帯域を使用している状態のトラフィック可視化デモ
- ルーターのQoS統計画面でパケット優先制御が動作していることを確認するデモ

### 図解・テロップ素材

- QoS（Quality of Service）の概念図（帯域をバケツに見立てた優先キューのイメージ）
- DiffServ / DSCPの仕組みを示すパケットヘッダ図
- 配信環境での推奨QoS設定一覧表（OBS: 最優先 / ゲーム: 高 / 通常通信: 普通）
- 主要ルーターメーカー別のQoS機能比較表

## 必要機材

### 所有済み

- PC1 / PC2（Windows DSCP設定デモ用）

### 追加購入（必須）

- QoS対応ルーター（ASUS RT-AX86U Pro / TP-Link Archer AX73）: 15,000-25,000円
  - 本エピソードの主題そのもの。ルーター管理画面でのQoSルール作成操作デモが必要
  - トラフィックモニター画面のキャプチャも必要
  - ASUS / TP-Linkのどちらかであれば、主要メーカーの比較表も実機ベースで制作可能
