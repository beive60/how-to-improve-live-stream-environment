# EP009 / VT-01: VTube Studioパラメータスムージング - 機械的な動きを排除するチューニング

## 必要な動画素材

### 画面キャプチャ

- VTube Studioの設定画面（トラッキングパラメータ一覧）
- スムージング値の調整スライダー画面（各パラメータ個別の設定）
- VTube Studioのキャリブレーション画面
- Live2Dモデルのパラメータ設定画面（VTube Studio内のモデル設定）

### デモ映像

- スムージング値が低すぎる場合のデモ（カクカクした機械的な動作、細かいジッターが目立つ）
- スムージング値が高すぎる場合のデモ（動きが遅延してもっさり、口パクが遅れる）
- 最適なスムージング値に調整した後のデモ（自然で滑らかな動き）
- 顔の各部位（目、口、首）のスムージング値を個別に調整する操作デモ
- ビフォーアフター並列比較映像（画面を左右分割して同一動作を並列表示）
- 素早い動き（激しいリアクション）と穏やかな動きの両方でのテスト映像

### 図解・テロップ素材

- スムージングの処理原理図（移動平均 / 補間処理のイメージ）
- パラメータ別の推奨スムージング値一覧表（目 / 口 / 顔回転 / 体の動き）
- スムージング値 vs 応答速度のトレードオフを示すグラフ

## 必要機材

### 所有済み

- Live2Dモデル: Beive（カスタム）推奨。マッピング数20でデモ映えが良い
- PC2: RTX 3060（VTube Studio動作用）
