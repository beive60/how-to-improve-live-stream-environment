# EP026 / DO-04: Streamer.bot演出自動化 - OBS WebSocket連携による配信演出の自動制御

## 必要な動画素材

### 画面キャプチャ

- Streamer.botのメイン画面（アクション一覧・イベント一覧）
- Streamer.botのアクション編集画面（サブアクションの構成）
- OBS WebSocket連携の設定画面（接続先・認証情報の設定）
- チャットイベントトリガーの設定画面（Twitch/YouTube チャット連動の条件設定）
- タイマー・スケジュール設定画面（定期実行アクションの設定）
- 変数・条件分岐の設定画面（C#コードアクション）

### デモ映像

- Streamer.botのインストールと初期設定の手順デモ
- OBS WebSocket接続を確立する設定操作デモ
- チャットコマンドでOBSシーンを切り替える演出デモ（コメント入力 → シーン変化）
- チャンネルポイント交換でサウンドエフェクトを再生する演出デモ
- フォロー通知でアラートアニメーションを表示する演出デモ
- 複数のサブアクションを連鎖させた複合演出デモ（音声 + 映像エフェクト + テロップ）
- 条件分岐を含むアクション（例: 特定ユーザーのみ反応する演出）のデモ

### 図解・テロップ素材

- Streamer.botのアーキテクチャ図（イベントソース → トリガー → アクション → OBS/音声/チャット）
- イベント → トリガー → アクションの対応関係表（代表的な演出パターン一覧）
- Streamer.bot vs StreamElements vs Nightbot の機能比較表
- 演出自動化ワークフローのシーケンス図
