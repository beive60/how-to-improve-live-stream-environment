# EP015 / NE-02: CBRビットレート設計 - ネットワーク変動に対するマージン計算

## 必要な動画素材

### 画面キャプチャ

- OBS出力設定画面（配信タブ > ビットレート / レート制御: CBR の設定）
- OBS出力設定画面（録画タブ > ビットレート設定の比較用）
- ブラウザベース速度テスト結果画面（上りスループット・ジッター値の表示）
- iPerf等のネットワーク帯域計測ツールの実行結果画面
- YouTube Live / Twitchの推奨ビットレート表のWebページキャプチャ

### デモ映像

- ビットレートが回線帯域に対して高すぎる場合のデモ（フレームドロップ・映像途切れ）
- ビットレートが低すぎる場合のデモ（映像にブロックノイズ・ぼやけが発生）
- 適切なビットレート設定時の安定した配信映像デモ
- ネットワーク帯域の時間変動を速度テストで複数回計測する操作デモ
- ジッター（帯域の揺れ幅）の計測結果をもとにマージンを計算する手順デモ

### 図解・テロップ素材

- CBRビットレート計算式の図解（回線上り帯域 x マージン係数 = 設定ビットレート上限）
- 帯域とジッターの関係を示す時系列グラフ（実測値にマージンラインを重畳）
- 解像度・フレームレート別の推奨ビットレート早見表（720p30 / 1080p30 / 1080p60）
- CBR vs VBR vs CQP の特性比較表
