# EP002 / AE-02: VST処理チェーン構築入門 - EQ/コンプレッサー/リミッターで配信音声を整える

## 必要な動画素材

### 画面キャプチャ

- OBSの音声フィルタ追加画面（フィルタ一覧 > VST 2.x プラグイン選択）
- VSTプラグインUI画面（ReaEQ / TDR Nova 等のEQ）
- VSTプラグインUI画面（ReaComp / TDR Kotelnikov 等のコンプレッサー）
- VSTプラグインUI画面（Limiter No6 / ReaXcomp 等のリミッター）
- OBSの音声フィルタチェーン全体の並び順

### デモ映像

- VST未処理の生マイク音声（環境音・低域こもり・音量ばらつきが出ている状態）
- EQ適用後の音声変化デモ（低域カット・中高域ブーストの効果）
- コンプレッサー適用後の音量均一化デモ（小声とシャウトの差が縮まる様子）
- リミッター適用後のピーク制御デモ（クリッピング防止の効果）
- 処理チェーン全段適用後の最終出力デモ（ビフォーアフター比較）
- 各パラメータを極端に設定した場合の悪い例（過剰EQ、潰しすぎコンプ等）

### 図解・テロップ素材

- 音声処理チェーンのフロー図（マイク入力 → EQ → コンプ → リミッター → 出力）
- 各プラグインの役割を一言で示す対照表
- EQの周波数帯ごとの役割図（低域・中域・高域と配信音声への影響）
- コンプレッサーの基本パラメータ図解（Threshold / Ratio / Attack / Release）

## 必要機材

### 所有済み

- マイク: Audio-Technica AT-2020
- オーディオIF: Roland VT-4（VSTホストとしてはOBSを使用するため、IF側の制約なし）
- PC2: RTX 3060（OBS + VSTプラグインの動作確認用）

### 追加購入

なし。本エピソードはソフトウェア完結（無料VSTプラグイン: ReaEQ, TDR Nova, ReaComp, TDR Kotelnikov, Limiter No6, ReaXcomp）。
