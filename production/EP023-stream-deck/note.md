# EP023 / DO-01: Stream Deck / Touch Portalマクロ構築 - ワンオペ配信の効率化

## 必要な動画素材

### 画面キャプチャ

- Stream Deckアプリのボタン設定画面（アクション割り当て、アイコン設定）
- Stream Deckアプリのマルチアクション設定画面（複数操作の逐次実行）
- Stream Deckアプリのプロファイル管理画面
- Touch Portalのページ設計画面（ボタン配置・アクション設定）
- Touch Portalのイベント/条件設定画面
- OBS WebSocketプラグインとの連携設定画面

### デモ映像

- Stream Deck本体の実写映像（ボタンのLCD表示、押下操作）
- ボタン1つでシーン切替が実行されるデモ（配信画面の変化とStream Deckの操作を同時に表示）
- マクロ実行デモ: シーン切替 + BGM変更 + テロップ表示を1ボタンで同時実行
- 音声ミュート/ミュート解除のトグル操作デモ
- Touch Portalをスマートフォン/タブレットで操作する実写デモ
- プロファイルの切替デモ（配信中 / 休憩中 / エンディング等の状況別ボタン配置）
- マクロの設定手順を一から構築する操作デモ

### 図解・テロップ素材

- 配信ワークフロー図（手動操作 vs マクロ化後の操作ステップ比較）
- Stream Deck vs Touch Portalの機能比較表（価格 / 物理ボタン / カスタマイズ性）
- おすすめボタン配置レイアウト図（配信シーン別の推奨マクロ一覧）
- ワンオペ配信における操作効率化の前後比較表

## 必要機材

### 所有済み

- Elgato Stream Deck（本エピソードの主要デモ機材）
- PC2（OBS連携デモ用）

### 追加購入

なし。Stream Deckを所有済み。Touch Portalはスマートフォンアプリ（無料/有料）で対応。
