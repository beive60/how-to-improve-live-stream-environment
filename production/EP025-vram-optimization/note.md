# EP025 / DO-03: VRAM Allocation最適化 - ゲームとOBSのGPUリソース配分

## 必要な動画素材

### 画面キャプチャ

- タスクマネージャーのGPUタブ（プロセス別のGPUメモリ使用量・GPU使用率）
- GPU-Z のセンサータブ（VRAM使用量、GPU Load、Memory Controller Load の時系列表示）
- NVIDIAコントロールパネルの3D設定画面（アプリケーション別の設定）
- OBSのエンコーダー設定画面（NVENC設定: プリセット、Look-ahead、B-Frames等）
- OBSの出力設定画面（解像度スケーリング、FPS設定）
- ゲーム内のグラフィック設定画面（VRAM使用量表示があるゲームの例）

### デモ映像

- ゲーム単体起動時のVRAM使用量・GPU使用率の計測デモ
- ゲーム + OBS（NVENC）起動時の計測デモ
- ゲーム + OBS + VTube Studio 起動時の計測デモ
- VRAM不足でパフォーマンスが低下する状態のデモ（スタッター発生、テクスチャ品質低下）
- ゲーム側のグラフィック設定を調整してVRAM使用量を削減する操作デモ
- OBSエンコーダー設定の最適化操作デモ（VRAM消費を抑えつつ画質を維持）
- 最適化前後のFPSとVRAM使用量の比較計測デモ

### 図解・テロップ素材

- GPU内部のリソース競合図（ゲームレンダリング / OBSエンコード / AI処理がVRAMを取り合うイメージ）
- VRAM使用量の実測結果棒グラフ（各条件別の内訳）
- NVENC設定パラメータがVRAM消費に与える影響の対照表
- GPU世代別のVRAM容量と推奨設定の早見表（6GB / 8GB / 12GB / 16GB）

## 必要機材

### 所有済み

- PC2: NVIDIA GeForce RTX 3060（VRAM 12GB。NVENCエンコード、GPU-Zモニタリングに対応）
- PC1: AMD Radeon RX 7900 XT（NVIDIA非搭載環境との比較解説に活用可能）

### 追加購入

なし。本エピソードはPC2のRTX 3060（VRAM 12GB）で全デモを実施可能。GPU-Z、タスクマネージャー等の計測ツールは無料。
